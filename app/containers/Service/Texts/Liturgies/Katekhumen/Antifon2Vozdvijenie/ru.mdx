### Второй антифон, глас 2

~~1~~ Боже, отчего / Ты нас навеки отверг?

**Припев** Распятый по пло́ти Сын Божий, / спаси нас, поющих Тебе: / аллилуия!

~~2~~ Воспомни народ, который Ты избрал издре́вле.

**Припев** Распятый по пло́ти Сын Божий, / спаси нас, поющих Тебе: / аллилуия!

~~3~~ Воспомни го́ру Сион, где поселился Ты.

**Припев** Распятый по пло́ти Сын Божий, / спаси нас, поющих Тебе: / аллилуия!

~~4~~ Бог — от века мой Царь, спасение Он совершил посреди земли.

**Припев** Распятый по пло́ти Сын Божий, / спаси нас, поющих Тебе: / аллилуия!

**Слава, и ныне.** О единородный Сын и Слово Бога, / Ты, бессмертный, ради нашего спасения добровольно воплоти́вшийся / от святой Богородицы и Вечноде́вы Марии, /  неизменно вочелове́чившийся / и распя́тый, Христе Боже, Смертию Смерть попра́вший, / один из Святой Троицы, // прославляемый вместе с Отцом и Святым Духом, — спаси нас!
