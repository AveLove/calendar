### Третий антифон, глас 7

~~1~~ Наде́ющиися на Го́спода, / я́ко гора́ Сио́н, не подви́жится во век.

**Припев** Преобрази́лся еси́ на горе́, Христе́ Бо́же, / показа́вый ученико́м Твои́м сла́ву Твою́, / я́коже можа́ху, / да возсия́ет и нам, гре́шным, / Свет Твой присносу́щный / моли́твами Богоро́дицы, // Светода́вче, сла́ва Тебе́.

~~2~~ Го́ры о́крест Его́, / и Госпо́дь о́крест люде́й Свои́х от ны́не и до ве́ка.

**Припев** Преобрази́лся еси́ на горе́, Христе́ Бо́же, / показа́вый ученико́м Твои́м сла́ву Твою́, / я́коже можа́ху, / да возсия́ет и нам, гре́шным, / Свет Твой присносу́щный / моли́твами Богоро́дицы, // Светода́вче, сла́ва Тебе́.

~~3~~ Го́споди, кто обита́ет в жили́щи Твое́м? / Или́ кто всели́тся во святу́ю го́ру Твою́?

**Припев** Преобрази́лся еси́ на горе́, Христе́ Бо́же, / показа́вый ученико́м Твои́м сла́ву Твою́, / я́коже можа́ху, / да возсия́ет и нам, гре́шным, / Свет Твой присносу́щный / моли́твами Богоро́дицы, // Светода́вче, сла́ва Тебе́.

~~4~~ Кто взы́дет на го́ру Госпо́дню? / Или́ кто ста́нет на ме́сте святе́м Его́?

**Припев** Преобрази́лся еси́ на горе́, Христе́ Бо́же, / показа́вый ученико́м Твои́м сла́ву Твою́, / я́коже можа́ху, / да возсия́ет и нам, гре́шным, / Свет Твой присносу́щный / моли́твами Богоро́дицы, // Светода́вче, сла́ва Тебе́.
