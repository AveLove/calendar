### Третий антифон, глас 4, Пс 109:1-3

~~1~~ Рече́ Госпо́дь Го́сподеви моему́: / седи́ одесну́ю Мене́.

**Припев** Рождество́ Твое́, Христе́ Бо́же наш, / возсия́ ми́рови свет ра́зума, / в нем бо звезда́м служа́щии / звездо́ю уча́хуся / Тебе́ кла́нятися, Со́лнцу Пра́вды, / и Тебе́ ве́дети с высоты́ восто́ка. // Го́споди, сла́ва Тебе́.

~~2~~ До́ндеже положу́ враги́ Твоя́ / подно́жие ног Твои́х.

**Припев** Рождество́ Твое́, Христе́ Бо́же наш, / возсия́ ми́рови свет ра́зума, / в нем бо звезда́м служа́щии / звездо́ю уча́хуся / Тебе́ кла́нятися, Со́лнцу Пра́вды, / и Тебе́ ве́дети с высоты́ восто́ка. // Го́споди, сла́ва Тебе́.

~~3~~  Жезл си́лы по́слет Ти Госпо́дь от Сио́на, / и госпо́дствуй посреде́ враго́в Твои́х.

**Припев** Рождество́ Твое́, Христе́ Бо́же наш, / возсия́ ми́рови свет ра́зума, / в нем бо звезда́м служа́щии / звездо́ю уча́хуся / Тебе́ кла́нятися, Со́лнцу Пра́вды, / и Тебе́ ве́дети с высоты́ восто́ка. // Го́споди, сла́ва Тебе́.

~~4~~  С Тобо́ю нача́ло в день си́лы Твоея́, / во све́тлостех святы́х Твои́х.

**Припев** Рождество́ Твое́, Христе́ Бо́же наш, / возсия́ ми́рови свет ра́зума, / в нем бо звезда́м служа́щии / звездо́ю уча́хуся / Тебе́ кла́нятися, Со́лнцу Пра́вды, / и Тебе́ ве́дети с высоты́ восто́ка. // Го́споди, сла́ва Тебе́.
