**Стихира на «Хвалите», воскресная, глас 4**

**Н** О всесильный Господи, / Крест и Смерть претерпевший и воскресший из мёртвых, // мы прославляем Твоё Воскресение!

**Стихира на «Хвалите», воскресная, глас 4**

**Н** Отеческого лона не покидая, / Ты, единородное Слово Божье, / пришёл на землю по Своему человеколюбию / и стал Человеком неизменно. / Ты претерпел по пло́ти Крест и Смерть, / по божеству непричастный страданию. / Восстав же из мёртвых, / Ты дарова́л бессмертие человеческому роду, // ибо один Ты всесилен.

**Стихира на «Хвалите», воскресная, глас 4**

**Н** Ты, Спаситель, при́нял по пло́ти смерть, / чтобы добыть нам бессмертие, / и во гроб вселился Ты, / дабы освободить нас из ада, воскресив вместе с Собою. / Ты пострадал как Человек, / но воскрес как Бог. / Посему мы восклицаем: / «О Господь, Податель Жизни, слава Тебе, // один столь человеколюбивый!»
