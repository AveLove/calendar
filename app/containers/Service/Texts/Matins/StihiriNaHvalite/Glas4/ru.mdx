**Стихиры на «Хвалите», воскресения, глас 4**  
**Н** О всесильный Господи, / Крест и Смерть претерпевший и воскресший из мёртвых, // мы прославляем Твоё Воскресение!

**Стихиры на «Хвалите», воскресения, глас 4**  
**Н** Неразлучно пребывая в недрах Отца, / Ты, единородное Слово Божье, / пришёл на землю по Своему человеколюбию / и стал человеком неизменно. / Ты претерпел по пло́ти Крест и Смерть, / по божеству непричастный страданию. / Восстав же из мёртвых, / Ты дарова́л бессмертие человеческому роду, // ибо один Ты всесилен.

**Стихиры на «Хвалите», воскресения, глас 4**  
**Н** Ты, Спаситель, при́нял по пло́ти смерть, / чтобы добыть нам бессмертие, / и во гроб вселился Ты, / дабы освободить нас из ада, воскресив вместе с Собою. / Ты пострадал как Человек, / но воскрес как Бог, / и посему мы восклицаем: / «О Господь, Податель Жизни, слава Тебе, // один столь человеколюбивый!»
