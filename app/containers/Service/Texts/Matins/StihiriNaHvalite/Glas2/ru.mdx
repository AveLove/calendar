**Стихира на «Хвалите», воскресная, глас 2**

**Н** Всё, что дышит, и всё творение / Тебя, Господи, прославляет, / ибо Ты Крестом упразднил Смерть / и явил всем народам Твоё Воскресение из мёртвых, // один столь человеколюбивый.

**Стихира на «Хвалите», воскресная, глас 2**

**Н** Пусть иудеи ответят: / ка́к воины, стерегущие Царя, упустили Его / и почему камень **[**гроба**]** не удержал Скалу Жизни? / Или пусть они предъявят погребённого, / или пусть покло́нятся Воскресшему, / с нами возглашая: // «Слава Твоему великому состраданию, Спаситель наш, слава Тебе!»

**Стихира на «Хвалите», воскресная, глас 2**

**Н** Возрадуйтесь, народы, и возликуйте, / ибо ангел, си́дя на камне у гроба, / сам возвестил нам радостную весть, говоря: / «Христос, Спаситель мира, воскрес из мёртвых / и наполнил весь мир благоуханием! // Возрадуйтесь же, народы, и возликуйте!»

**Стихира на «Хвалите», воскресная, глас 2**

**Н** Ангел прежде Твоего, Господи, зачатия / возвестил Благодатной «Радуйся!»; / ангел же при Твоём Воскресении / от славного Твоего гроба камень отвалил. / Один вместо скорби благовествовал зна́мения радости, / другой же вместо смерти возвещал нам Владыку, дару́ющего Жизнь. / Посему мы возглашаем Тебе: // «О Благодетель всех **[**`людей`**]**, Господи, слава Тебе!»

**Стихира на «Хвалите», воскресная, глас 2**

**Н** Ангел прежде Твоего, Господи, зачатия / возвестил Благодатной «Радуйся!»; / ангел же при Твоём Воскресении / от славного Твоего гроба камень отвалил. / Один вместо скорби благовествовал зна́мения радости, / другой же вместо смерти возвещал нам Владыку, дару́ющего Жизнь. / Посему мы возглашаем Тебе: // «О Благодетель всех **[**`людей`**]**, Господи, слава Тебе!»

**Стихира на «Хвалите», воскресная, на «И ныне», богородичен, глас 2**

**Н** Преблагослове́нна ты, Богородица Дева, / ибо Воплоти́вшийся от тебя ад пленил, / Адама воззвал, проклятие сокрушил и Еву освободил, / Смерть умертвил и нас оживотворил. / Посему мы воспеваем и восклицаем: // «Благословен Христос, Бог **[**наш**]**, так благоволи́вший, слава Тебе!»
