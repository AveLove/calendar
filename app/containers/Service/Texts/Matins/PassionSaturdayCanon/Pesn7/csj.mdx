**Ирмос** Неизрече́нное чу́до, / в пещи́ изба́вивый / преподо́бныя о́троки из пла́мене, / во гро́бе мертв / бездыха́нен полага́ется, / во спасе́ние нас пою́щих: / Изба́вителю Бо́же, // благослове́н еси́.
