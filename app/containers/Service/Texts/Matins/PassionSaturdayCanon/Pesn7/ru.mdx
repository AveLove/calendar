**Ирмос** О, неизъяснимое чудо! / Тот, Кто избавил святых отроков в печи́ от пламени, / полагается во гробе / мёртвым и бездыханным, / дабы спасти нас, воспевающих: // «Благословен ты, Боже, Избави́тель!»

Был уя́звлен в сердце ад, / приняв Того, Кто был в рёбра копьём уя́звлен, / и стенает, сожигаемый божественным огнём / во спасение нам, воспевающим: // «Благословен ты, Боже, Избави́тель!»

Благословен *сей* гроб! / Ведь он при́нял в себя Создателя, словно бы спящего, / и явился божественной сокровищницей Жизни / во спасение нам, воспевающим: // «Благословен ты, Боже, Избави́тель!»

По чину уме́рших, / Жизнь всех *людей* претерпевает положение во гроб / и являет его источником Воскресения, / дабы спасти нас, воспевающих: // «Благословен ты, Боже, Избави́тель!»

Во аде, во гробе и в Эдеме / божество Христово / пребывало единым и неразлучным со Отцом и Духом / во спасение нам, воспевающим: // «Благословен ты, Боже, Избави́тель!»
