**Ирмос** Тебя, подвесившего на вода́х / всю землю без опоры, / увидев вися́щим / на лобном месте, / творение в великом ужасе вострепетало, восклицая: // «Никто не свят, кроме Тебя, Господи!»

Ты во многих видениях явил *пророкам* образы погребения Твоего. / Ныне же как Бог и Человек Ты, Владыка, открыл Свои тайны / и тем, кто пребывает во аде и восклицает: // «Никто не свят, кроме Тебя, Господи!»

Ты руки распростёр и соединил некогда разделённое. / Утеснённый же плащаницей и гробом, / Ты от оков освободил пленников, взывающих: // «Никто не свят, кроме Тебя, Господи!»

В гробнице запечатанной Ты был заперт, о Невместимый, / ибо *так* Ты явил силу Свою в божественном действии / тем, кто воспевает: // «Никто не свят, кроме Тебя, Господи!»

*Катавасия* Тебя, подвесившего на вода́х / всю землю без опоры, / увидев вися́щим / на лобном месте, / творение в великом ужасе вострепетало, восклицая: // «Никто не свят, кроме Тебя, Господи!»
