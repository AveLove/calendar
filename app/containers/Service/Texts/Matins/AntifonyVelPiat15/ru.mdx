**Антифон 13-й, глас 6**

Собрание иудеев просило Пилата / распять Тебя, Господи! / Вины́ в Тебе не найдя, / они виновного Вара́вву освободили, / а Тебя, Праведного, осудили, / грех скверного убийства получив в удел. / Но **[**не**]** воздай им, Господи, // тщетное против Тебя замыслившим!

Кого страшится и трепещет всё **[**творение**]**, / и всякий язык воспевает — / Христа, Божию Силу и Божию Премудрость, — / священники били по щека́м / и давали Ему желчь. / И всё претерпел Ты, / Своею Кровью желая спасти нас от беззаконий наших // как человеколюбивый.

**Слава, богородичен** О Богородица, которая через слово, что превыше **[**всякого**]** разума, / родила своего Создателя, // моли Его о спасении душ наших.

**И ныне** О Богородица, которая через слово, что превыше **[**всякого**]** разума, / родила своего Создателя, // моли Его о спасении душ наших.

**Антифон 14-й, глас 8**

Господи, Ты взял Себе в спутники разбойника, / руки осквернившего кровью! / Сопричти́ к Себе с ним и нас // как благой и человеколюбивый.

Разбойник, малое слово вымолвив на кресте, / великую веру обрёл, / в один миг спасён был / и первым, врата отворив, вошёл в рай. / Приня́вший его покаяние // Господи, слава Тебе!

**Слава, богородичен** Радуйся, от ангела приня́вшая радость для мира! / Радуйся, родившая своего Творца и Господа! // Радуйся, удостоившаяся стать Матерью Божией!

**И ныне** Радуйся, от ангела приня́вшая радость для мира! / Радуйся, родившая своего Творца и Господа! // Радуйся, удостоившаяся стать Матерью Божией!

**Антифон 15-й, глас 6**

Ныне То́т висит на древе, / Кто подвесил землю среди вод; / венцом из терния уве́нчан ангелов Царь; / в плащ багряный облачён Одевающий небо облаками; / пощёчины воспри́нял во Иордане Освободивший Адама; / гвоздя́ми пригвождён Жених Церкви; / пронзён копьём Сын Девы. / Поклоняемся страданиям Твоим, Христе! / Поклоняемся страданиям Твоим, Христе! / Поклоняемся страданиям Твоим, Христе! — // Яви же нам и Твоё славное Воскресение!

Не как иудеи будем праздновать, / ибо Пасха наша — Христос, за́кланный за нас. / Очистим же себя от всякой скверны / и в чистоте помолимся Ему: / «Восстань, Господи, спаси нас // как человеколюбивый!»

Крест Твой, Господи, / для Народа Твоего есть жизнь и воскресение, / и, надеясь на него, / мы Тебя, распятого Бога нашего, воспеваем. — // Помилуй нас!

**Слава, богородичен** Видя Тебя, Христе, вися́щим **[**на кресте**]**, / родившая Тебя взывала: / «Что́ есть тайна непостижимая, которую вижу я, о Сын мой? Ка́к Ты умираешь, плотью пригвождённый на древе, // о Жизни Податель?»

**И ныне** Видя Тебя, Христе, вися́щим **[**на кресте**]**, / родившая Тебя взывала: / «Что́ есть тайна непостижимая, которую вижу я, о Сын мой? Ка́к Ты умираешь, плотью пригвождённый на древе, // о Жизни Податель?»

**Седален, глас 4**

Ты выкупил нас из проклятья закона / драгоценною Кровью Своей. / Ко кресту пригвождённый / и копьём пронзённый, / Ты людям бессмертье излил. // О Спаситель наш, слава Тебе!
