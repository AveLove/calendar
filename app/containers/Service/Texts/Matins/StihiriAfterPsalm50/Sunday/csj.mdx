**Слава.** Моли́твами Апо́столов, / Ми́лостиве, очи́сти // мно́жества согреше́ний на́ших.

**И ныне.** Моли́твами Богоро́дицы, / Ми́лостиве, очи́сти // мно́жества согреше́ний на́ших.

Поми́луй мя́, Бо́же, / по вели́цей ми́лости Твое́й / и по мно́жеству щедро́т Твои́х // очи́сти беззако́ние мое́.

Воскре́с Иису́с от гро́ба, / я́коже прорече́, / даде́ на́м живо́т ве́чный // и ве́лию ми́лость.
