**Степенна, антифон 1, глас 5**

**Н** Внегда́ скорбе́ти мне, / Дави́дски пою́ Тебе́, Спа́се мой: // изба́ви ду́шу мою́ от язы́ка льсти́ваго.

Пусты́нным живо́т блаже́н есть, // Боже́ственным раче́нием воскриля́ющимся.

**Слава.** Святы́м Ду́хом одержа́тся вся, / ви́димая же с неви́димыми: / Самодержа́вен бо сый, // Тро́ицы Еди́н есть нело́жно.

**И ныне, тойже.**

**Степенна, антифон 2, глас 5**

**Н** На го́ры душе́, воздви́гнемся, // гряди́ та́мо, отню́дуже по́мощь и́дет.

Десна́я Твоя́ рука́, и мене́, Христе́, каса́ющися, // от ле́сти вся́кия да сохрани́т.

**Слава.** Свято́му Ду́ху богосло́вяще рцем: / Ты еси́ Бог, Живо́т, Раче́ние, Свет, Ум: // Ты Благосты́ня, Ты ца́рствуеши во ве́ки.

**И ныне, тойже.**

**Степенна, антифон 3, глас 5**

**Н** О ре́кших мне: во дворы́ вни́дем Госпо́дня: // ра́дости мно́гия испо́лнен быв, моли́твы возсыла́ю.

В дому́ Дави́дове стра́шная соверша́ются: // огнь бо та́мо паля́ всяк сра́мный ум.

**Слава.** Свято́му Ду́ху живонача́льное досто́инство, / от Него́же вся́кое живо́тно одушевля́ется, // я́ко во Отце́, ку́пно же и Сло́ве.

**И ныне, тойже.**
