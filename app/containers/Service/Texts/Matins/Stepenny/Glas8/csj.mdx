**Степенна, антифон 1, глас 8**

**Н** От ю́ности моея́ враг мя искуша́ет, сластьми́ пали́т мя: // аз же наде́яся на Тя, Го́споди, побежда́ю сего́.

Ненави́дящии Сио́на, / да бу́дут у́бо пре́жде исторже́ния я́ко трава́: // ссе́чет бо Христо́с вы́я их, усече́нием мук.

**Слава.** Святы́м Ду́хом, Е́же жи́ти вся́ческим: / Свет от Све́та, Бог Вели́к: // со Отце́м пое́м Ему́, и с Сло́вом.

**И ныне, тойже.**

**Степенна, антифон 2, глас 8**

**Н** Се́рдце мое́ стра́хом Твои́м да покры́ется смиренному́дрствующее: // да не возне́сшееся отпаде́т от Тебе́, Всеще́дре.

На Го́спода име́вый наде́жду, не устраши́тся тогда́, // егда́ огне́м вся суди́ти и́мать, и му́кою.

**Слава.** Святы́м Ду́хом, всяк кто боже́ственный ви́дит, и предглаго́лет, / чудоде́йствует вы́шняя, в Трие́х Еди́наго Бо́га поя́: // а́ще бо и трисия́ет, единонача́льствует Божество́.

**И ныне, тойже.**

**Степенна, антифон 3, глас 8**

**Н** Воззва́х Тебе́, Го́споди, вонми́, / приклони́ ми у́хо Твое́ вопию́щу, и очи́сти, // пре́жде да́же не во́змеши мене́ отсю́ду.

В ма́тери свое́й земли́ отходя́й всяк, па́ки разреша́ется, // прия́ти му́ки, или́ по́чести пожи́вших.

**Слава.** Святы́м Ду́хом богосло́вие, Еди́ница Трисвята́я: / Оте́ц бо Безнача́лен: от Него́же роди́ся Сын безле́тно, // и Дух Сопресто́лен, Сообра́зен, от Отца́ спросия́вший.

**И ныне, тойже.**

**Степенна, антифон 4, глас 8**

**Н** Се ны́не что добро́, или́ что красно́; но е́же жи́ти бра́тии вку́пе: // в сем бо Госпо́дь обеща́ живо́т ве́чный.

О ри́зе свое́й, И́же кри́ны се́льныя украша́яй, // повелева́ет, я́ко не подоба́ет пещи́ся.

**Слава.** Святы́м Ду́хом, Единови́дною вино́ю, / вся содержа́тся миропода́тельне: // Бог бо Сей есть, Отцу́ же и Сы́нови Единосу́щен Госпо́дственне.

**И ныне, тойже.**
