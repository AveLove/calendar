Увидев воскресение Христа, / покло́нимся святому Господу Иисусу, / одному безгрешному! / Кресту Твоему поклоняемся, Христе, / и святое Воскресение Твоё воспеваем и прославляем. / Ты — Бог наш: / кроме Тебя мы иного не знаем, / и имя Твоё призываем. / Приди́те, все верные, / покло́нимся святому Христову Воскресению, / ибо вот, через Крест радость пришла всему миру. / Всегда благословляя Господа, / мы воспеваем Воскресение Его, / ведь Он, распятие претерпев, // Смертию Смерть победил.

Светись же, светись, / новый Иерусалим, / ибо слава Господня над тобою взошла. / Ныне торжествуй и веселись, Сион, / а ты, чистая Богородица, радуйся // восстанию Рождённого тобой!

О Христос — / Пасха великая и святейшая, / Премудрость, Божье Слово и Сила! / Дару́й нам полнее к Тебе приобщаться // в невечерний день Царства Твоего!
