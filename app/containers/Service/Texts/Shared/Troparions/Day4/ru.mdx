#### Тропарь четверга, св. апостолов, глас 3

Святые апостолы, / хода́тайствуйте пред милостивым Богом, / о дарова́нии ду́шам нашим // согрешений прощения!

#### Тропарь четверга, свт. Николая, глас 4

Мерилом веры и образом кротости, / воздержности учителем / явила **(или:** явили**)** тебя пастве твоей / сама вещей истина **(или:** сами твои дела**)**. / Ты обрёл в смирении величие, / в нищете — богатство, / посему, иерарх и отец наш Николай, / хода́тайствуй пред Христом Богом // о спасении душ наших!
