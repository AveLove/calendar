#### Кондак четверга, св. апостолов, глас 3

Святые апостолы, / хода́тайствуйте пред милостивым Богом, / о дарова́нии ду́шам нашим // согрешений прощения!

#### Кондак четверга, свт. Николая, глас 4

Правилом веры и образом кротости, / воздержания учителем / явила **[или:** явили**]** тебя для паствы твоей / этих вещей истина **[или:** сами дела твои**]**. / Ты обрёл через смирение — величие, / через нищету — богатство, / посему, о иерарх и отец наш Николай, / хода́тайствуй пред Христом Богом // о спасении душ наших!
