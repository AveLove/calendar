#### Кондак субботы, мучеников, глас 8

Словно первые плоды людского рода, / Тебе, Господи, насадившему творение, / мир наш приносит богоносных мучеников. / По их молитвам и ради Богородицы, / сохрани в нерушимом мире // Церковь Твою **[**и Твоё общество**]**, о Многоми́лостивый!

#### Кондак субботы, заупокойный, глас 8

Со святыми упокой, / Христе, души чад **(или:** рабов**)** Твоих, / где нет ни страдания, ни скорби, ни стенания, // но Жизнь бесконечная.
