### Непорочны, статия 3

**Начинает снова со сладкопением хор, на глас 3-й:**

Все люди из рода в род / приносят песнь // при погребении Твоём, о Христос мой! 

~~132~~ Взгляни на меня и помилуй меня, как Ты делаешь с любящими имя Твоё.

Иосиф Аримафейский, с древа сняв / и плащаницею обвив, // во гробе Тебя полагает.

~~133~~ Пусть слово Твоё сделает твёрдым мой шаг, не дай беззаконию овладеть мною;

О Христос мой, / мироносицы пришли // миро принести Тебе с усердием.

~~134~~ избавь меня от притеснения людского, и буду хранить повеления Твои.

Приди, всё творение, // воспоём погребальные песни Творцу!

~~135~~ Будь благоскло́нен ко мне, рабу Твоему, и научи меня установлениям Твоим.

Вместе с мироносицами / мы мудро умасти́м миром // Живого, словно мёртвого!

~~136~~ Из глаз моих текут потоки слёз, ведь не исполняют люди Твой закон.

О трижды блаженный Иосиф, / предай погребению тело // Христа, Жизни Подателя!

~~137~~ Праведен Ты, о Господи, справедливы деяния Твои.

Те, кого Он питал манной, // готовы растоптать своего Благодетеля.

~~138~~ Наставления, что Ты заповедал, – правда и соверше́нная истина.

Те, кого Он питал манной, // приносят Спасителю желчь вместе с уксусом.

~~139~~ Негодование снеда́ет меня, оттого что враги мои забыли слова́ Твои.

О безумие! // Убийцы пророков убивают Христа!

~~140~~ Слово Твоё – словно очищенный в горниле металл, и я, раб Твой, люблю его.

Словно слуга безрассудный, // пре́дал ученик бездну Премудрости.

~~141~~ Я презре́н и ничтожен, но повелений Твоих не забыл.

Продав Освободителя, // остался пленником коварный Иуда.

~~142~~ Правда Твоя – правда вечная, и закон Твой – истина.

~~143~~ Скорби и бе́ды постигли меня; заповеди Твои – радость для меня.

~~144~~ Наставления Твои вовек справедливы, вразуми меня, чтобы я был жив.

~~145~~ Я взываю всем сердцем: ответь мне, Господи, и буду хранить установления Твои.

~~146~~ Призываю Тебя: спаси меня, и буду соблюдать наставления Твои.

**Слава. Троичен:** О Троица, мой Бог — / Отец, Сын и Дух, // помилуй мир!

**И ныне. Богородичен:** О Дева, / удостой чад твоих // увидеть восстание Сына твоего!
