### Тропарь Страстной седмицы, глас 8

Когда славные ученики́ / при омовении ног на Последней тра́пезе просветились, / тогда помрачился / одержи́мый сребролюбием нечестивый Иуда; / и Тебя, справедливого Судию́, пре́дали судьям беззаконным. / Смотри, любитель выгоды, / на вы́гадавшего себе верёвку; / берегись ненасытной души, / что решилась на такое против Учителя. // Всеблагой Господи, слава Тебе!
