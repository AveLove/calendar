Услышали пастухи, как ангелы воспевают Пришествие Христа во пло́ти, / и, устремившись к Нему, словно к Пастырю, видят Его, словно непорочного Агнца, / пастбище обретшего во утробе Марии, / и, её воспевая, воскликнули: / «Радуйся, Агнца и Пастыря Матерь, / радуйся, двор словесных овец. / Радуйся, ибо ты невидимых врагов отражаешь, / радуйся, ибо ты райские двери отверзаешь. / Радуйся, ибо небеса вместе с землёй ликуют, / радуйся, ибо люди с ангельским хором поют. / Радуйся, немолкнущие уста апостолов, / радуйся, непобедимое дерзновение мучеников. / Радуйся, твёрдое веры основание, / радуйся, пресветлое благодати зна́мение. / Радуйся, через кого ни с чем остался ад, / радуйся, через кого мы облеклись во славу. // Радуйся, Невеста неневестная!»
