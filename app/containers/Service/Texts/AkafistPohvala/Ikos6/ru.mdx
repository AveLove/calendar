Озарив Египет светом истины, Ты, Спаситель, рассеял тьму обмана, / ведь пали идолы египетские, не вы́неся силы Твоей. / Все же, избавленные от них, взывали к Богородице: / «Радуйся, людей восставление, / радуйся, бе́сов низвержение. / Радуйся, лживые козни поправшая, / радуйся, коварство идолов обличившая. / Радуйся, море, что потопило фараона духовного, / радуйся, скала, что напоила жаждущих жизни. / Радуйся, столп огненный, ведущий пребывающих во тьме, / радуйся, покров над миром, распростёртый шире облака. / Радуйся, пища, манны преемница, / радуйся, священного наслаждения служительница. / Радуйся, земля обетования, / радуйся, источающая молоко и мёд. // Радуйся, Невеста неневестная!»
