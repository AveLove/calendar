Дева, Бога во чреве вместившая, устремилась к Елизавете, / у той же во чреве младенец, познав приветствие Девы, возрадовался / и взыграл, словно песнь воспел, взывая к Богородице: / «Радуйся, Лоза отрасли неувядающей, / радуйся, обретение Пло́да непорочного. / Радуйся, ибо ты возделываешь человеколюбивого Земледельца, / радуйся, ибо ты порождаешь жизнь нашу Насадившего. / Радуйся, земля, приносящая урожай милосердия, / радуйся, престол изобильной жертвы умилостивления. / Радуйся, ибо ты луг наслаждения произращаешь, / радуйся, ибо ты пристань для душ уготовляешь. / Радуйся, приятный Богу фимиам хода́тайства, / радуйся, умилостивительное приношение всего мира. / Радуйся, к смертным Божье благоволение, / радуйся, смертных к Богу дерзновение. // Радуйся, Невеста неневестная!»
