Светоприе́мную свещу́, су́щим во тьме я́вльшуюся, зрим святу́ю Де́ву, / невеще́ственный бо вжига́ющи огнь, / наставля́ет к ра́зуму Боже́ственному вся, / заре́ю ум просвеща́ющая, зва́нием же почита́емая, си́ми: / Ра́дуйся, луче́ у́мнаго Со́лнца; / ра́дуйся, свети́ло незаходи́маго Све́та. / Ра́дуйся, мо́лние, ду́ши просвеща́ющая; / ра́дуйся, яко́ гром, враги́ устраша́ющая. / Ра́дуйся, я́ко многосве́тлое возсиява́еши просвеще́ние; / ра́дуйся, я́ко многотеку́щую источа́еши реку́. / Ра́дуйся, купе́ли живопису́ющая о́браз; / ра́дуйся, грехо́вную отъе́млющая скве́рну. / Ра́дуйся, ба́не, омыва́ющая со́весть; / ра́дуйся, ча́ше, че́рплющая рáдocть. / Ра́дуйся, обоня́ние Христо́ва благоуха́ния; / ра́дуйся, животе́ та́йнаго весе́лия. // Ра́дуйся, Неве́сто Неневе́стная.
