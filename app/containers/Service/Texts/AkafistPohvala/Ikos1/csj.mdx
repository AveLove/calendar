А́нгел предста́тель с небесе́ по́слан бысть рещи́ Богоро́дице: ра́дуйся, / и со безпло́тным гла́сом воплоща́ема Тя зря, Го́споди, ужаса́шеся / и стоя́ше, зовы́й к Ней такова́я: / Ра́дуйся, Е́юже рáдocть возсия́ет; / ра́дуйся, Е́юже кля́тва изче́знет. / Ра́дуйся, па́дшаго Ада́ма воззва́ние; / ра́дуйся, слез Е́виных избавле́ние. / Ра́дуйся, высото́ неудобовосходи́мая челове́ческими по́мыслы; / ра́дуйся, глубино́ неудобозри́мая и а́нгельскима очи́ма. / Ра́дуйся, я́ко еси́ Царе́во седа́лище; / ра́дуйся, я́ко но́сиши Нося́щаго вся. / Ра́дуйся, звездо́, явля́ющая Со́лнце; / ра́дуйся, утро́бо Боже́ственнаго воплоще́ния. / Ра́дуйся, Е́юже обновля́ется тварь; / ра́дуйся, Е́юже покланя́емся Творцу́. // Ра́дуйся, Неве́сто Неневе́стная.
