**Ирмос** Отроков от печи́ Избавивший, / став Человеком, / страдает, словно смертный, / и чрез Его страдание наша смертная плоть / облекается в красоту нетления. / Он один благословен — // Бог отцов, неизреченно славный.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Жёны богомудрые / с миром за Тобою устремились, / и Кого они, словно смертного, со слеза́ми искали, / Тому, радуясь, / поклонились как живому Богу, / и тайну Пасхи, о Христос, // Твоим ученикам возвестили.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Смерти празднуем мы умерщвление, / ада разрушение, / иной, вечной, Жизни начало / и, ликуя, воспеваем / это Свершившего, / одного благословенного — // Бога отцов, неизреченно славного.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Воистину эта спасительная ночь — / священная, всепраздничная / и светозарная, / она — предвестница / светоносного дня Воскресения. / В эту ночь негасимый Свет // из гроба во плоти всем воссиял.
