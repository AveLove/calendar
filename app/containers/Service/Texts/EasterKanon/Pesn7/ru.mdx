**Ирмос** Отроков от печи́ Избавивший, / став Человеком, / страдает, словно смертный, / и Своим страданием / облекает смертных в красоту нетления. / Он один благословен и препрославлен — // Бог отцов *наших*.

**Припев** Христос воскрес **(или:** восстал<R>)</R> из мёртвых.

Жёны богомудрые / с миром за Тобою устремились, / и, Кого они искали со слеза́ми, словно смертного, / Тому они, радуясь, поклонились, как живому Богу / и Того как Пасху сокровенную // Твоим, Христос, ученикам возвестили.

**Припев** Христос воскрес **(или:** восстал<R>)</R> из мёртвых.

Мы празднуем Смерти умерщвление, ада разрушение, / иной, вечной, Жизни начало / и, ликуя, воспеваем это Совершившего, / ибо Он один благословен и препрославлен — // Бог отцов *наших*.

**Припев** Христос воскрес **(или:** восстал<R>)</R> из мёртвых.

Воистину эта спасительная ночь — / священная, праздничная и светозарная, / она — светоносного дня Воскресения предвестница. / В эту ночь негасимый Свет // из гроба во плоти всем воссиял.
