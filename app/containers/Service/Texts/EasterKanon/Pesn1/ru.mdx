**Ирмос** Воскресения день, / да просветимся, верные! / Пасха! Господня Пасха! / От смерти к жизни / и с земли на Небеса / Христос Бог привёл нас, // воспевающих победную песнь:

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Очистим чувств пятерицу — / и у́зрим Христа, / блистающего / неприступным светом Воскресения, / и явственно да услышим Его, / говорящего: «Радуйтесь!», // воспевая победную песнь:

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Небеса, торжествуя, да радуются, / а земля да ликует, / пусть празднует весь мир, / видимый же и невидимый, / ведь Христос восстал — // Веселие вечное!

**Катавасия** Воскресения день, / да просветимся, верные! / Пасха! Господня Пасха! / От смерти к жизни / и с земли на Небеса / Христос Бог привёл нас, // воспевающих победную песнь.

Христос воскрес **(или:** восстал**)** из мёртвых, / Смертию Смерть поправ // и тем, кто во гробах, жизнь дарова́в! **(Трижды,	поско́ру.)**
