**Ирмос** Сошёл Ты в преисподнюю, о Христос, / и сокрушил засо́вы вечные, / заключавшие узников, / и, словно Иона из чрева кита́, // Ты на третий день восстал из гроба.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Из гробa Ты восстал, о Христос, / сохранив печати целыми, / и в рождении Своём / Ты не нарушил девства Матери, // и отверз нам райские двери.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

О Спаситель мой, / Ты как Бог ─ живая и незаколаемая жертва; / Ты Себя по воле Своей принёс Отцу; / и, восстав из гроба, / с Собой воздвиг Адама, // отца всех людей.

**Катавасия** Сошёл Ты в преисподнюю, о Христос, / и сокрушил засо́вы вечные, / заключавшие узников, / и, словно Иона из чрева кита́, // Ты на третий день восстал из гроба.

Христос воскрес **(или:** восстал**)** из мёртвых, / Смертию Смерть поправ // и тем, кто во гробах, жизнь дарова́в! **(Трижды, поско́ру.)**
