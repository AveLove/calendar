**Ирмос** Сошёл Ты в преисподнюю, Христе, / и сокрушил засо́вы вечные, заключавшие узников, / и, словно из кита́ Иона, // Ты на третий день восстал из гроба.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Из **[**опечатанной**]** гробницы / Ты восстал, Христе, / сохранив печати целыми, / и в рождении Своём / Ты не нарушил девства Матери, // и отверз нам райские двери.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

О Спаситель мой, / Ты как Бог ─ живая и незаколаемая жертва; / Ты Себя по воле Своей принёс Отцу; / и, восстав из гроба, / с Собой воздвиг Адама, // отца всех людей.

**Катавасия** Сошёл Ты в преисподнюю, Христе, / и сокрушил засо́вы вечные, заключавшие узников, / и, словно из кита́ Иона, // Ты на третий день восстал из гроба.

Христос воскрес **(или:** восстал**)** из мёртвых, / Смертию Смерть поправ // и тем, кто во гробах, жизнь дарова́в! **(Трижды, поско́ру.)**
