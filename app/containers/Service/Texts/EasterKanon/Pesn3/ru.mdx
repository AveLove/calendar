**Ирмос** Придите, питие испием новое, / не чудом из скалы бесплодной изводимое, / но [Сам] Источник нетления — Христа, / ведь, как дождь, / Он излился из гроба, // в Нём мы и утверждаемся.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Ныне всё исполнилось светом: / небо, и земля, / и преисподняя! / Да празднует же всё творение / Воскресение Христово, // в нём мы и утверждаемся.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Вчера меня с Тобою погребали, Христе, — / сегодня же восстаю / с Тобой, воскресшим; / вчера меня с Тобою распинали — / приобщи же меня, о Спаситель, // к Твоей Славе во Царствии Твоём!
