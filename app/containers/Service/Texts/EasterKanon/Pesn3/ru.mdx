**Ирмос** Придите, будем пить новое питие, / не чудом из скалы бесплодной изводимое, / но Сам Источник нетления, / ведь, как дождь, излился из гроба Христос, // в Нём же мы и утверждаемся.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Ныне всё исполнилось света: / небо, и земля, / и преисподняя! / Да празднует же всё творение / Воскресение Христово, // в нём же мы и утверждаемся.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Вчера меня с Тобою погребали, о Христос, — / сегодня же восстаю / с Тобой, воскресшим; / вчера меня с Тобою распинали — / приобщи же меня, о Спаситель, // к Твоей славе во царствии Твоём!

**Катавасия** Придите, будем пить новое питие, / не чудом из скалы бесплодной изводимое, / но Сам Источник нетления, / ведь, как дождь, излился из гроба Христос, // в Нём же мы и утверждаемся.

Христос воскрес **(или:** восстал**)** из мёртвых, / Смертию Смерть поправ // и тем, кто во гробах, жизнь дарова́в! **(Трижды, поско́ру.)**
