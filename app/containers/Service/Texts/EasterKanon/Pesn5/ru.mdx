**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Узники ада, / видя Твоё, Христе, безмерное благосердие, / устремились к Свету весёлыми ногами, // прославляя Пасху вечную.

**Припев** Христос воскрес **(или:** восстал**)** из мёртвых.

Со свети́льниками **[**в рука́х**]** / приблизимся ко Христу, / из гробницы, подобно Жениху, выходящему, / и вместе с ликующими чина́ми **[**ангелов**]** / будем праздновать // Пасху Божью спасительную!
