**Припев** Величает душа моя / воскресшего на третий день из гроба // Христа, Жизни подателя!

**Ирмос** Светись же, светись, / новый Иерусалим, / ведь Слава Господня над тобою взошла; / ныне торжествуй и веселись, Сион; / а ты, чистая Богородица, радуйся // восстанию Рождённого тобой!

**Припев** Христос — новая Пасха, / жертва живая, / Агнец Божий, // уносящий грехи мира!

**Тропарь** Голос Твой, Христе, ─ / божественный, любимый и сладчайший, / ведь Ты обещал непреложно / быть с нами до скончания века. / Имея сей голос как якорь надежды, // мы, верные, радуемся!

**Припев** Ангел воскликнул Благодатной: / чистая Дева, радуйся! / И снова скажу: радуйся! / Твой Сын восстал / в третий день из гроба / и воздвиг мёртвых – // люди, возвеселитесь!

**Тропарь** О Христос — / Пасха великая и святейшая, / Премудрость, Божье Слово и Сила! / Дару́й же нам / полнее к Тебе приобщаться // в невечерний день Царства Твоего!

**Катавасия** Светись же, светись, / новый Иерусалим, / ведь Слава Господня над тобою взошла; / ныне торжествуй и веселись, Сион; / а ты, чистая Богородица, радуйся // восстанию Рождённого тобой!

Христос воскрес **(или:** восстал**)** из мёртвых, / Смертию Смерть поправ // и тем, кто во гробах, жизнь дарова́в! **(Трижды, поско́ру.)**
