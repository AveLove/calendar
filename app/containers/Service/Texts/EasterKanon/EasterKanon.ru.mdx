**Припев** Христос воскрес [восстал] из мёртвых, Смертию Смерть поправ и тем, кто во гробах, жизнь даровав!

**Песнь 1, ирмос**  
Воскресения день, / да просветятся люди! / Пасха! Господня Пасха! / Ведь от смерти к жизни / и от земли — на Небеса / Христос Бог нас перевёл, // победную песнь воспевающих.

**Песнь 1, тропарь 1**  
Очистим чувства и увидим Христа, / блистающего неприступным светом Воскресения, / и явственно от Него услышим: «Радуйтесь!», // победную песнь воспевая.

**Песнь 1, тропарь 2**  
Так пусть Небеса достойно радуются, / земля же да ликует, / пусть празднует весь мир, видимый и невидимый, / ведь восстал Христос — // веселие вечное!

**Песнь 3, ирмос**  
Придите, питие испейте новое, / не чудом из скалы бесплодной изводимое, / но от источника нетления, / что дождём пролилось из гроба Христа, — // в Нём и утверждаемся.

**Песнь 3, тропарь 1**  
Ныне всё исполнилось светом: / небо, и земля, и преисподняя. / Пусть же празднует всё творение / Воскресение Христа, // Коим оно и утверждается.

**Песнь 3, тропарь 2**  
Вчера меня с Тобою погребали, Христе, — / сегодня восстаю с Тобой, воскресшим;/ вчера меня с Тобою распинали — / прославь же и меня с Собой, Спаситель, // когда Ты придёшь царствовать.

**Ипакои́, глас 8**  
Жёны, пришедшие с Марией до рассвета,/ найдя́ камень отва́ленным от гроба,/ услышали от ангела: «Что вы ищете, как человека, среди мёртвых/ во Свете вечносущем обитающего?/ Ви́дите — вот погребальные пе́лены;/ спешите и миру возвести́те, что Господь восстал и умертвил Смерть,/ ибо Он — Сын Бога, спасающего род человеческий».

**Песнь 4, ирмос**  
На стражу, ожидая Бога, / возвещающий Бога Аввакум, / да встанет с нами / и укажет светоносного ангела, / громко восклицающего: / «Ныне спасение миру, / ибо воскрес Христос // как всесильный!»

**Песнь 4, тропарь 1**  
Первенцем Девы явился Христос. / Будучи человеком, / [Он] Агнцем именуется, / а непорочным — / как не имеющий пятна. / 
Он — наша Пасха, / и как Бог истинный // назван совершенным.

**Песнь 4, тропарь 2**  
Словно однолетний ягнёнок, / Христос за всех был за́клан по воле Своей: / Он — Пасха очищающая, / Он нам благословенный Венец! / И снова из гроба прекрасное // Правды Солнце нам воссияло.

**Песнь 4, тропарь 3**  
Пра́оотец Давид / пред сенью ковчега скакал и играл, / мы же, как святой народ Божий, / видя исполнение зна́мений, / да возрадуемся в Духе, / ибо восстал Христос // как всесильный.

**Песнь 5, ирмос**  
Пробудимся же рано поутру́, / и вместо мира [благовонного] / принесём Владыке песнь, / и увидим Христа — / Солнце Правды, // всем Жизнь излучающего!

**Песнь 5, тропарь 1**  
Пу́тами ада уде́рживаемые, / увидев Твоё, Христе, безмерное благосердие, / мы устремились к Свету весёлыми ногами, // прославляя Пасху вечную.

**Песнь 5, тропарь 2**  
Со свети́льниками в рука́х приблизимся ко Христу, / из гробницы, подобно Жениху, выходящему, / и вместе с ликующими чина́ми ангелов будем праздновать: // Пасха Божья — спасение!

**Песнь 6, ирмос**  
Сошёл Ты в преисподнюю, Христе, / и сокрушил засо́вы вечные, / заключавшие узников, / и, словно из кита́ Иона, // из гроба на третий день восстал.

**Песнь 6, тропарь 1**  
Из опечатанной гробницы / Ты восстал, Христе, сохранив печати целыми, / и в рождении Своём не нарушил Ты девства Матери, и так отверз нам // райские двери!

**Песнь 6, тропарь 2**  
О, Спаситель мой / Ты как Бог ─ живой и незаколаемый Агнец; / Ты Себя добровольно принёс Отцу; / и, восстав из гроба, // воздвиг с Собою Адама всеродного.

**Кондак, глас 8**  
Хотя и сошёл Ты во гроб, о Бессмертный, / Ты низложи́л силу ада; / и восстал как победитель, Христе Боже, / жёнам-мироно́сицам сказав: «Радуйтесь!»; / и апостолам Твоим мир дарова́л; // падшим же пода́л воскресение.

**Икос, глас 8**  
Не́когда девы-мироносицы ожидали Солнце, / до захода солнца сошедшее во гроб, / взыскуя Его на рассвете, словно прихода дня, / и друг ко другу взывали: / «Пойдёмте, подруги, / ароматами умастим тело живоносное и погребённое, / плоть Того, Кто воскресил отпадшего Адама, лежащего в земле. / Пойдёмте, поспешим к Нему, как волхвы́, / и покло́нимся, и принесём благовония / в дар Тому, Кто уже не спелёнут, но саваном обёрнут, / и оплачем Его, и воскликнем: / “О Владыка, воскре́сни // и падшим подай воскресение!”»

**Песнь 7, ирмос**  
Отроков от печи́ избавивший, / став Человеком, / страждет, словно смертный, / и чрез страдание Его / смертное облекается в красоту нетления. / Он один благословенный Бог отцов, // более всех прославля́емый.

**Песнь 7, тропарь 1**  
Жёны богомудрые, с миррой за Тобою устремились, / и Кого они, словно смертного, со слеза́ми искали, / Тому, радуясь, поклонились / как живому Богу, / и тайну Пасхи // ученикам Твоим, о Христос, возвестили.

**Песнь 7, тропарь 2**  
Смерти празднуем умерщвление, / ада разрушение, / иной, вечной, Жизни начало / и, ликуя, воспеваем Виновника — / одного благословенного Бога отцов, // более всех прославля́емого!

**Песнь 7, тропарь 3**  
Воистину эта спасительная ночь — / священная, праздничная и светозарная, / она предвестница светоносного дня воскресения, / в которую вечный Свет // из гроба во плоти всем воссиял.

**Песнь 8, ирмос**  
Это и́збранный и святой день, / первый по субботе, / царственный и великий, / праздников праздник / и торжество из торжеств, // в который благословляем Христа вовеки!

**Песнь 8, тропарь 1**  
Приди́те, приобщимся / к новому пло́ду лозы виноградной — / к божественной радости / в нарочитый день Воскресения, / к Царству Христа, // воспевая Его как Бога вовеки.

**Песнь 8, тропарь 2**  
Возведи очи твои, о Сион, и посмотри: / вот чада твои, как божественно сияющие звёзды, / пришли к тебе от запада, востока, севера и моря // благословить в тебе Христа вовеки.

**Песнь 8, тропарь 3**  
Отец Вседержитель, и Слово, и Дух, / в трёх лицах единое естество, / сверхсущностное и пребожественное, / в имя Твоё мы крестились // и Тебя благословляем во все веки.

**Припев 9й песни**  
Величит душа моя воскресшего на третий день из гроба Христа, Жизни подателя!

**Песнь 9, ирмос**  
Светись же, светись, / новый Иерусалим, / ведь Слава Господня над тобою взошла; / ныне торжествуй и веселись, Сион; / а ты, чистая Богородица, радуйся // Воскресению Рождённого тобой!

**Песнь 9, тропарь 1**  
О сколь божественно, любезно и сладостно / Твоё, Христе, слово, / ведь Ты дал неложное обещание / быть с нами до скончания века / Имея его якорем надежды, // мы, верные, радуемся.

**Песнь 9, тропарь 2**  
О Христос — / Пасха великая и святейшая, / Премудрость, Божье Слово и Сила! / Дару́й нам полнее к Тебе приобщаться // в невечерний день Царства Твоего.

**Эксапостиларий**  
Плотию уснув, / как смертный, / о Царь и Господь, / Ты на третий день восстал, / Адама воздвигнув из тьмы / и упраздни́в Смерть. / Ты — Пасха бессмертия, // мира спасение!
