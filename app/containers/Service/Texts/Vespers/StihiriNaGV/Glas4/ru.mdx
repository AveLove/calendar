**Стихира на «Господи, взываю», воскресная, глас 4**

**Н** Животворящему Кресту Твоему / мы, Христе Боже, непрестанно поклоняемся / и Твоё тридневное Воскресение прославляем, / ведь через него обновил Ты, Всесильный, / разрушенную природу человека / и показал нам путь на Небеса, // как один благой и человеколюбивый.

**Стихира на «Господи, взываю», воскресная, глас 4**

**Н** Добровольно пригвождённый к крестному древу, / Ты, Спаситель, упразднил наказание, / со времён древа непослушания бывшее; / и, сойдя во ад, о Сильный, / Ты как Бог разбил оковы Смерти. / Посему мы поклоняемся Твоему Воскресению из мёртвых, / с радостью возглашая: // «Всесильный Господи, слава Тебе!»

**Стихира на «Господи, взываю», воскресная, глас 4**

**Н** Врата ада сокрушил Ты, Господи, / и Своею Смертью царство Смерти разрушил, / род же человеческий от погибели освободил, / миру дарова́в жизнь, и бессмертие, // и милость великую.

**Стихира на «Господи, взываю», воскресная, на «Слава, и ныне» (если есть стихира Минеи или Триоди на «Слава», то на «И ныне»), богородичен, глас 4**

**Н** Пророк Давид, через тебя, Богородица, ставший пра́отцем Божьим, / воспев псалом перед Свершившим для тебя великое, / пророчески предвозвестил о тебе: / «Стала Царица одесну́ю Тебя!» / Ведь Матерью, родившей **[**нашу**]** Жизнь, соделал тебя Христос Бог, / возжелавший вочеловечиться от тебя без отца, / дабы повреждённый страстями Свой образ возродить, / и, заблудившуюся в гора́х овцу отыскав, / взять её на плечи и к Отцу принести, / соединить по Своей воле с небесными силами и спасти мир, — // ибо милость Его велика и неистощима.
